# 2024-02-27 更新日志

# 🎉 新功能

- 短信發送記錄查詢功能
    - 支持查看短信發送記錄，包括發送時間、發送狀態、發榮場景、發送狀態說明、發送內容等常用信息
    - 路徑：「客戶管理系統」-「消息中心」-「消息記錄」
    - 權限：message.log_query，消息發送記錄查詢[敏感權限]，該功能權限需單獨授權後方可執行查詢操作

    - 路徑：「客戶管理系統」-「消息中心」-「消息記錄」
    - 權限：message.log_query，消息發送記錄查詢[敏感權限]，該功能權限需單獨授權後方可執行查詢操作
